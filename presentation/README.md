# Презентация проекта

В этой директории находятся два файла:

- `Semantic_Map_BoE_template.pptx` — рабочий шаблон, который используется как источник оформления;
- `Semantic_Map_BoE_public.pptx` — финальная публичная версия презентации.

Для пересборки итоговой колоды используется скрипт `scripts/build_presentation.py`.

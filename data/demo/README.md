# Синтетический demo-набор

Файл `demo_articles.csv` создан автором проекта специально для публичного smoke-test сценария.

Он:

- не относится к корпусу UKTwitNewsCor;
- не содержит текстов из стороннего корпуса;
- предназначен только для проверки работоспособности кода и структуры pipeline.

Результаты, полученные на demo-наборе, не следует интерпретировать как исследовательские выводы.
